Syntax Analyzer

Recursive-descent parser validates declarations and arithmetic expressions.

parser.cpp
Grammar Rules:
program → statement*
statement → decl_stmt | assign_stmt
decl_stmt → type IDENTIFIER [= expr] ;
expr → term ((+ | -) term)*
term → factor ((* | /) factor)*
factor → NUMBER | IDENTIFIER | ( expr )
